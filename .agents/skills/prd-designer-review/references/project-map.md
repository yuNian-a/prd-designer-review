# Project Map

用于在需求预审时识别当前需求所属的项目、业务线与产品类型，帮助 AI 判断需求的背景范围和设计上下文。

---

## 使用说明

在执行需求预审时，AI 应查阅本文件，了解：
- 当前需求属于哪个产品线
- 该产品线的目标用户和核心业务场景
- 需求评审时应重点关注的业务特性
